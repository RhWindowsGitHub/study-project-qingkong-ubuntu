## About

StudyProjectForQingKongUbuntu
### 前后端分离项目模板

> 包含基本的登陆、注册、密码重置等功能，可以二次开发编写具体场景下的应用程序
> 

* 登陆功能（支持用户名和邮箱登陆）
* 注册用户 （通过邮箱注册）
* 重置密码 （通过邮箱重置密码）
